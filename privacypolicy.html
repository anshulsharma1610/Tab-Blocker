Privacy Policy – New Tab Blocker

The New Tab Blocker Chrome extension is designed to give users a smoother, distraction-free browsing experience by blocking specific websites from opening new tabs or pop-ups. We value your privacy and are committed to ensuring your data stays completely private.

Information Collection and Use

This extension does not collect, store, or transmit any personal data. All configuration and user preferences (such as blocked websites) are stored locally in your browser using Chrome’s built-in chrome.storage.sync API. This storage is synchronized securely with your Google account but is never accessed or shared externally.

Permissions

The permissions requested by this extension are the minimum necessary for its single purpose:

  • tabs – to detect when new tabs are opened.
  • storage – to save and sync your blocked site list.
  • webNavigation – to check if a blocked site opens new tabs.
  • host permissions – to allow the extension to function on all sites without reading or modifying content.

Data Sharing

No information is ever transmitted to any external servers or shared with third parties. The extension functions entirely within your local browser environment.

Remote Code

This extension does not use or load any remote code. All scripts are bundled within the extension package submitted to the Chrome Web Store.

Changes to This Policy

If updates are made to this policy, they will be reflected on this page with the date of the latest revision.
